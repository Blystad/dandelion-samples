Basic styling

This example shows how to apply some style on the cells.

In this example:

  • The City column is centered using cssStyle="text-align:center;" , which update the "th" tag of the column.
  • All the body cells of the Mail column are centered using cssCellStyle="text-align:center;" whereas a CSS class has been applied on the "th" cell using cssClass="myCustomClass".
